# User Roles

## Student

- View Daily Ta'lim.
- Listen to lessons.
- Complete quizzes.
- Track personal progress.

## Parent

- View linked family learners.
- Review child progress and assigned lessons.
- Receive future notifications and family reflections.

## Teacher

- View assigned classes.
- Review learner progress.
- Manage lesson delivery and class planning.
- Create draft content in future versions.

## Admin

- Manage madrasahs, teachers, users, lessons, and assets.
- Review operational metrics.
- Control publishing and content workflows.
- Access audit-ready administrative views.

## Super Admin

- Manage organizations and tenants.
- Configure platform-wide settings.
- Access cross-madrasah reporting.
