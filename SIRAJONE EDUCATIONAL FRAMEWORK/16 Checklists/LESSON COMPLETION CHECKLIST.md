## THE SIX-LAYER COMPLETION STANDARD

A lesson is not complete until it satisfies all six layers:

| Layer | Component | Check |
|-------|-----------|-------|
| 1 | Reader Pages (all page types written and correct) | ✅ / ❌ |
| 2 | Companion (all 9 sections, correct key) | ✅ / ❌ |
| 3 | Profiles (relevant figures have profiles in app) | ✅ / ❌ |
| 4 | Maps (relevant locations have map entries) | ✅ / ❌ |
| 5 | References (sources documented in References Library) | ✅ / ❌ |
| 6 | Discovery Links (ClosingPage discoveryNote points to real content) | ✅ / ❌ |

**A lesson is declared complete only when all 6 layers are confirmed.**

---

## LAYER 1 — READER PAGES CHECKLIST

- [ ] PartDividerPage: title, subtitle, description written
- [ ] Segment 1: virtue, ḥadīth, sections written
- [ ] Segment 2: virtue, ḥadīth, sections written
- [ ] Segment 3: virtue, ḥadīth, sections written
- [ ] Segment 4 (if applicable): virtue, ḥadīth, sections written
- [ ] ReflectionPage: 4–5 questions written
- [ ] ClosingPage: paragraphs, Duʿāʾ, nextLessonPreview, discoveryNote written
- [ ] No future lesson content narrated
- [ ] All honorifics correct
- [ ] All transliteration correct
- [ ] TypeScript compiles with 0 errors

## LAYER 2 — COMPANION CHECKLIST

- [ ] `lessonTitle` matches PartDividerPage title
- [ ] `whyThisMatters` written (contemporary relevance, not summary)
- [ ] `memoryGem` written with verified Tier 1 citation and exact ḥadīth number
- [ ] `oneMinuteSummary` written (no future lesson content)
- [ ] `keyFacts` written (4–7 entries with scholarly notes)
- [ ] `whatWeLearns` written (3–4 virtue entries)
- [ ] `familyDiscussion` written (4 questions with facilitatorNotes)
- [ ] `quickReview` written (5–6 Q&A pairs)
- [ ] `furtherReading` written (3–6 real obtainable works)
- [ ] Exported in lessonEnrichmentData.ts as `"sirah_journey:[N]": lessonN`
- [ ] Companion renders correctly in the live app

## LAYER 3 — PROFILES CHECKLIST

- [ ] All major historical figures in the lesson have profiles in the app's Profiles tab
- [ ] Profiles include: name, lineage, role, key facts, significance
- [ ] DiscoveryNote in ClosingPage references the correct profile(s) where applicable

## LAYER 4 — MAPS CHECKLIST

- [ ] All relevant geographical locations in the lesson have entries in the app's Maps tab
- [ ] Map entries include: location name, historical significance, connection to this lesson
- [ ] DiscoveryNote in ClosingPage references the correct map entry/entries where applicable

## LAYER 5 — REFERENCES CHECKLIST

- [ ] All Tier 1–2 sources used in the lesson are documented in the References Library (08 References Library)
- [ ] The lesson's research questions and resolutions are documented in the Research Archive (07 Research Archive)
- [ ] Any scholarly disagreements and their resolutions are documented in the Editorial Decisions Log (09)

## LAYER 6 — DISCOVERY LINKS CHECKLIST

- [ ] DiscoveryNote points to Profiles, Maps, or References content that actually exists
- [ ] DiscoveryNote does not instruct users on how to use the UI
- [ ] DiscoveryNote describes the educational benefit of exploring that content
